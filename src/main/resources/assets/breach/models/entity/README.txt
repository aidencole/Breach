WRONG FOLDER for GeckoLib models.

Do not put fallen_body_wide.geo.json here. This path is for vanilla block/item JSON models.

Move or copy your file to:

  src/main/resources/assets/breach/geckolib/models/entity/fallen_body_wide.geo.json

Then rebuild the mod. GeckoLib only loads .geo.json files from the geckolib/models/ tree.
